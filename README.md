## 简介 | Introduction

**Ai划词翻译** 摒弃了传统脚本陈旧的“工程师审美”，采用现代化的 ** (Glassmorphism)** UI 设计。

它旨在为你提供丝滑、纯粹的阅读翻译体验。支持 OpenAI、Claude、Gemini 以及国内的硅基流动 (SiliconFlow) 等多种主流大模型 API。

**无需安装庞大的浏览器插件，一个轻量级的脚本即可搞定一切。**

## ✨ 核心特性 | Features

### 🎨 现代化高颜值 UI
* **毛玻璃效果**: 采用 Backdrop Filter 实现磨砂质感，完美适配深色/浅色网页背景。
* **柔和视觉**: 精心调教的阴影与圆角设计，视觉体验极佳。
* **极简交互**: 拒绝屏幕污染，没有常驻悬浮球遮挡视线。

### 🤖 多接口支持
无缝衔接主流 AI 能力，支持自定义配置：
* **OpenAI API**: 兼容官方及所有支持 OpenAI 格式的中转服务。
* **Claude API**: 支持 Anthropic 格式接口。
* **Gemini API**: 支持 Google Gemini 系列模型。
* **硅基流动 (SiliconFlow)**: 完美支持国内高速稳定的官方接口。

### ⚡ 便捷操作体验
* **即划即现**: 选中并释放鼠标后，翻译图标自动浮现。
* **自由拖拽**: 翻译面板支持任意拖拽，不遮挡原文。
* **智能关闭**: 点击面板外任意空白处自动关闭。
* **一键复制**: 头部集成复制按钮，快速提取译文。

### 防废话
内置经过严格调优的 **System Prompt**，强制 AI 忽略闲聊指令，只输出纯粹的翻译结果，节省 Token 并提高阅读效率。

## 安装方法 | Installation

1.  **安装管理器**: 确保你的浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) (油猴) 扩展。
2.  **一键安装**: 点击下方按钮即可直接安装脚本。

[![安装脚本](https://img.shields.io/badge/👉%20点击安装脚本-Install-success?style=for-the-badge&logo=tampermonkey)](https://raw.githubusercontent.com/xiaobaiapp/ai-selection-translator/main/OpenAI_Translator.user.js)

## 配置指南 | Configuration

安装完成后，你需要进行简单的配置才能使用：

1.  刷新任意网页。
2.  点击浏览器右上角扩展栏的 **Tampermonkey** 图标。
3.  在菜单中找到 `Ai划词翻译` 脚本。
4.  点击脚本下方的 **`⚙️ 配置后台`** 菜单项。
5.  在弹出的精美面板中进行设置：
    * 选择你的 **API 供应商** (Provider)。
    * 填写对应的 **API Key**。
    * 填写 **接口地址** (Base URL)。
6.  点击保存即可生效。
